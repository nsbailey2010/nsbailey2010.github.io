Picture Scramble

This is the second version of Picture Scramble. The first version made use of the HTML5 data transfer API to switch the tiles, which relied on a dragStart event to begin the transfer. While this approach worked great on a PC, it proved to be problematic for mobile or touch devices since the dragStart was depedant on a mouse click event. To solve this issue, I used the JQuery UI library and Touch Punch which did an excellent job of providing touch functionality. Both versions make use of JavaScript and JQuery to generate the puzzle in an empty HTML div element called “board” when the page loads. The source code for each version is available through my GitHub repository.

Quote Generator

I created this quote generator as one of the zipline challenges from FreeCodeCamp. It was my first hands on experience creating an interface that communicated with a third party API. The app makes a request to Quotes on Design for a random quote and receives the quote in the form of a JSON object. The data is then formatted and displayed in the textbox on the user interface where it can then be tweeted by clicking the tweet button.

Local Weather

This app makes use of geolocation to determine the users coordinates and then makes a call to the OpenWeather API to retrieve the local weather information in the form of a JSON object. It then displays the information along with a background image and icon appropriate to the weather conditions. Images are courtesy of Pixabay and the icons are from the Weather Icons open-source icon font project.

Wikipedia Search

This app uses the Wikipedia API to retrieve the first ten results related to the users search criteria. Through creating this app, I learned a great deal about how API’s work as well as query formulation to retrieve only the desired information. The MediaWiki and Wikipedia API’s offer an impressive amount of flexibility and search customization options.
